# CalculadoraIMC

Calculadora de IMC (Índice de massa corporal) + estado nutricional

Calculadora simples criada em C# com intuito de validar estrutura e automação de cálculo de IMC para nutricionista responsável no projeto de desenvolvimento de sistema de atendimento.

Aplicativo de console utilizando C# e parâmetros simples para cálculo

Informações de cálculo meramente ilustrativos
